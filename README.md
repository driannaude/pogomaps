# Pokemon Go Maps

## Instructions
**MAC/LINUX:**

* `pip install -r requirements.txt`
* Put Google Maps API key in `config.json`
* Syntax: `./run.sh [user] [pass] "[location]"`
* Open browser to "http://localhost:8000"

**WINDOWS GUIDE:**
